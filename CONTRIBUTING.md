# Contributing to FolioLang

Thanks for your interest in contributing to FolioLang.

## How to contribute

1. Fork this repository.
2. Create a new branch from `main`.
3. Make your changes.
4. Open a Pull Request.
5. Wait for review before merging.

## Rules

- Do not commit directly to `main`.
- Keep changes small and focused.
- Explain what changed in the Pull Request description.
- Add examples when changing the `.folio` syntax.
- Update documentation when needed.
